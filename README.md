# kolhlashon
פרויקט קיוסק קול הלשון על עמדות לינוקס
===================


שלום.
זהו דף הפרויקט לבניית מערכת קיוסק לעמדת קול הלשון.
המערכת בשלבי תיכנון כרגע. ואשמח אם תצטרפו אלי ל‬[מייל](mailto:kolhlinux@gmail.com)

הרעיון העומד מאחורי הפרויקט הוא חיסכון בעלויות העמדות, ואבטחה טובה יותר.
כל זה ניתן להשגה על ידי שימוש בלינוקס כמערכת הפעלה, ועל .גבי מיני מחשבים כדוגמת ראספברי פי ודומייהם
הפצת קיוסק מהאתר [porteus-kiosk.org](porteus-kiosk.org)
היתרונות שבהפצה זו הם מהירות העלייה, ועלייה מאימג.
שליטה מרחוק בעידכונים.

---

תיכנון 
-------------

הפצות הקיוסק מיועדות להצגת לשונית דפדפן בודדת שהוגדרה מראש. 
לפי כך, הכי קל הוא לבנות דף אינטרנט מקומי שירוץ על המחשב המקומי ויבצע את התקשורת מול שרתי קול הלשון ואת ההעתקה לנגנים.
